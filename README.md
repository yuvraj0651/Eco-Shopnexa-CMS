## 📖 Overview

Eco Shopnexa CMS is a production-inspired Ecommerce Content Management System designed to manage products, orders, users, content sections, analytics, and store operations from a single dashboard.

The project follows a scalable architecture with reusable components, centralized state management, responsive design principles, and modern frontend development practices.

It is built to simulate a real-world SaaS ecommerce administration platform with enterprise-level UI patterns.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Eco-Ecom-CMS.git
```

## Move Into Project

```bash
cd Eco-Ecom-CMS
```

## Install Dependencies

```bash
npm install
```

## Run Frontend

```bash
npm run dev
```

---

# 🔥 Backend Setup

## Start JSON Server

```bash
json-server src/components/backend/db.json --port 5000
```

## Start Express Server

```bash
node server.js
```
